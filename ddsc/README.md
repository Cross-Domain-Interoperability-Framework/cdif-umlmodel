# README #

XMI representations of the UML model for the CDIF Data Description Profile

* cdif-ddsc_canonical-unique-names-eclipse.xmi - The model as Canonical XMI file with UML version 2.5 and XMI 2.5.1, and with unique association names for Eclipse
* cdif-ddsc_canonical-unique-names.xmi - The model as Canonical XMI file with UML version 2.5 and XMI 2.5.1, and with unique association names
